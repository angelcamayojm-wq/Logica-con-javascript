# 🚀 JavaScript Logic Lab

Proyecto web con **5 ejercicios de JavaScript**, correspondientes a las actividades **11 a la 15**.

Este proyecto permite practicar lógica básica usando arreglos, ciclos y funciones en JavaScript. 🧠⚡

---

## 📌 Ejercicios incluidos

| Ejercicio | Descripción |
|---|---|
| 🧮 Ejercicio 11 | Suma de elementos de vector con ciclo `for`. |
| 🔎 Ejercicio 12 | Encontrar el valor máximo en un array. |
| 🕵️ Ejercicio 13 | Búsqueda en array con ciclo `while`. |
| 🔁 Ejercicio 14 | Invertir un array con ciclo `for`. |
| 🔢 Ejercicio 15 | Contar números pares usando `for`. |

---

## 🛠️ Tecnologías usadas

- 🌐 HTML
- 🎨 CSS
- ⚙️ JavaScript
- 🚀 GitHub Actions
- 📄 GitHub Pages

---

## ▶️ Cómo usar el proyecto

1. Abre la página principal.
2. Selecciona uno de los ejercicios.
3. Ingresa los datos solicitados.
4. Presiona el botón del ejercicio.
5. Mira el resultado en pantalla. ✅
